# Sonlarni ekranga chiqarish

Sonlarni ham cout orqali ekranga chiqarishingiz mumkin.

Matndan farqli sonlarni qo'shtirnoqqa olmaymiz:

```cpp
#include <iostream>
using namespace std;

int main() {
  cout << 3;
  return 0;
}
```

Shuningdek, siz matematik hisob-kitoblarni bajarishingiz mumkin:

```cpp
cout << 3 + 3;
```

```cpp
cout << 2 * 5;
```
